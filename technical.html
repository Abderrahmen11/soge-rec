Intervention Workflow

Follow the standard process for handling interventions

1. Receive Assignment

Ticket is assigned to you by admin. Check details and requirements.

getAssignedTickets() SELECT * FROM tickets WHERE technician_id = X
2. Schedule Intervention

Plan your visit according to priority and availability.

checkSchedule() UPDATE tickets SET scheduled_date
3. Start Intervention

Begin working on the task and update the status.

startIntervention(ticket_id) UPDATE interventions SET status='in progress'
4. Update Progress

Regularly update notes and status as you work.

updateIntervention() UPDATE interventions SET notes/status
5. Complete Intervention

Mark as completed when all work is done.

completeIntervention() UPDATE interventions SET status='completed'
6. Notify Stakeholders

System automatically notifies client and admin.

generateNotification() INSERT INTO notifications
